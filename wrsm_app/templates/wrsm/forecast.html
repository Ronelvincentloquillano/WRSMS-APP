What is the forecast based on?

Data source: Completed sales for each customer only (Sales records and their created_date). Pending or open orders are not used in this calculation.

Days frequency: For each customer, all sale dates are sorted from oldest to newest. The system takes the number of whole calendar days between each pair of consecutive sales, then uses the average of those gaps (rounded). If there are fewer than two sales, there is no gap to average—frequency stays empty or zero.

Last ordered date: The calendar date of the most recent sale for that customer.

Next order date: Last ordered date plus days frequency (simple forward projection). This is a planning hint, not a guarantee—it does not model holidays, seasonality, or delivery schedules.

Use the sync button above to recompute from the latest sales history.
